Coffee Sales Excel Dashboard 

📊 Project Overview

This project analyzes coffee sales data using Microsoft Excel to uncover sales trends, customer behavior, and product performance. The dataset was cleaned, transformed, and analyzed, and the results were visualized through an interactive Excel dashboard.

🎯 Objectives
- Analyze coffee sales performance
- Identify top-selling coffee types
- Explore customer purchasing behavior
- Visualize sales trends through an interactive dashboard

🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- INDEX & MATCH
- VLOOKUP
- IF Functions
- Data Cleaning Techniques
- Dashboard Design

Data Preparation

The following data preparation steps were performed:
- Cleaned and formatted raw data
- Checked for missing values
- Standardized data types
- Created calculated columns
- Used INDEX & MATCH and VLOOKUP to retrieve customer and product information
- Applied logical formulas using IF functions

Dashboard Features

The Excel dashboard provides insights through:
- Total Sales Overview
- Sales by Coffee Type
- Sales by Country
- Customer Purchase Analysis
- Interactive filtering using Slicers

🔍 Key Insights
- Identified top-performing coffee products
- Analyzed customer purchasing patterns
- Compared sales performance across different countries
